# supernote-nytc
Downloads tomorrow's NYT crossword into a designated Dropbox folder.

## Instructions:
1. Clone
2. Set DROPBOX_ACCESS_TOKEN, NYT_COOKIE, and SUPERNOTE_UPLOAD_PATH secrets
3. Profit!
